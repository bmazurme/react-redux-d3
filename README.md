### Installation
Clone the repository on your computer:

`git clone ...`

Install dependencies: `npm install`

Run dev mode: `npm run dev`

Build project: `npm run build`

Clear project: `npm run clear`

Run eslint: `npm run eslint`

Launch: `npm start`
